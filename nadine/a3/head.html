HTML Guide

Common HTML Tags

Table

Tag Definition Examples
<!DOCTYPE html> A declaration that tells the browser what version of HTML the document is written in.
<html lang="en"> Tells the browser that all content on the page is English.
<meta charset="UTF-8"> Covers almost all of the characters and symbols in the world.
<title> Defines the name of the document.
<style> Internal CSS uses the tag <style> to define a unique style for a single HTML page.
<link rel="stylesheet" href="styles.css"> External CSS uses the style sheet tag to define the style for many HTML pages.
<link rel="icon" type="image/x-icon" href="/images/favicon.ico"> A favicon is an image is displayed to the left of the page title.
