Common HTML Tags outside <body>

Tag Definition Example
<!DOCTYPE>

Information to browser about what document type to expect

<!DOCTYPE html>
<html lang=""> Declares language of web page

<html lang="en">
<meta charset=""> Specifies character encoding for HTML document

<meta charset="UTF-8"
<title> Creates title on the tab on the browser

Title for this page: Assignment 3
<style> Defines the style for a single HTML page (Internal CSS)

This sentence is coloured orange with internal CSS.
<link rel="stylesheet" href="styles.css"> Defines the style for many HTML pages (External CSS)

This sentence is coloured yellow with external CSS.
<link rel="icon" type="image/x-icon" href="..."> Adds little picture (Favicon) next to page title on the browser tab

Favicon for this page:
<script> Embeds a client-side script (JavaScript)

The navigation bar on this page is coded with JavaScript
<noscript>

Displays alternate content to users that have disabled scripts in their browser or have a browser that doesn't support script

I.e. if scripts disabled this message is shown:Your browser does not support JavaScript!
<base>

Specifies the base URL and/or target for all relative URLs in a document

<base href="..." target="...">
