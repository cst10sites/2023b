Common HTML Tags outside <body>


Tag Definition Examples
<!DOCTYPE html> Doctype html is used to define what type of HTML is being used.
<html lang="en"> lang="..." is used to determine the language the website will be in.
<meta charset="UTF-8"> This tag makes almost every sign (characters in different languages), available.
<title> </title> This is the title of the website.
<style"> This tag goes inside of head, and it represents internal css. There is no internal css, like it was said in the instructions.
<link rel="stylesheet" href="style.css"> This is a link to a style sheet that contains external css. The green background was created in the external css.
<link rel="icon" type="image/x-icon" href="favicon.ico> This tag allows you to add an icon to the tab of your website. If you look at the top of this webpage, you can see this image.
