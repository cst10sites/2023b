HTML Guide

Common HTML Tags outside <body>

Tag Definition Examples
<!DOCTYPE html> This tag tells your browser what type of document the site is. All HTML documents have this tag at the very top of its code. View the source code of this site by pressing "Ctrl + U".

<html lang="en"></html> The <html> tag contains all the other tags of your html document other than the "!DOCTYPE" tag. the "lang="en"" part sets the language of your site to english. This sites language is English, or "en". Another language you could use is French, or "fr".

<meta charset="UTF-8"> This tag sets the character encoding of your site. UTF-8 contains characters for most of the languages and symbols that you could possibly need.

<title>…</title> The <title> tag allows you to create a title for your document. This sites title is "Assignment 3" and can be seen at the top of the browser.

<style></style> The <style> tag allows you to edit the internal CSS of your site, such as the shape, size, placement and colour of different aspects of your site.

[Sample Text]
<link rel="stylesheet" href="file name"> This tag tells the site to refer to another file to get the CSS from externally. This can be useful when trying to edit multiple pages of your site, by allowing you to just edit one CSS file rather than all documents. This site refers to the file "style.css" for its CSS.

<link rel="icon" type="image/x-icon" href="icon file name"> This tag allows you to have an icon for your site next to the title. This sites icon is a clipboard.
