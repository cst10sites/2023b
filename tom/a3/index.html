Tag Definition Examples
<h1> Defines a top-level heading <h1>Hello World</h1>
<p> Defines a paragraph <p>This is a paragraph.</p>
<a> Defines a hyperlink <a href="https://example.com">Visit Example.com</a>
<img> Defines an image <img src="image.jpg" alt="Description">
<ul> Defines an unordered list <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol> Defines an ordered list <ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
<li> Defines a list item <li>Item</li>
<div> Defines a division or a section <div>Content</div>
<span> Defines an inline container <span>Text</span>
<table> Defines a table
