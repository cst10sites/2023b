Tag Definition Examples
<!DOCTYPE html> Defines the document type <!DOCTYPE html>
<html lang="en"> Defines the language of the document <html lang="en">
<meta charset="UTF-8"> Defines the character encoding <meta charset="UTF-8">
<title> Defines the title of the document <title>Page Title</title>
<style> Describes internal CSS styles <style>body { color: red; }</style>
<link rel="stylesheet" href="styles.css"> Refers to an external CSS file <link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico"> Includes a favicon <link rel="icon" href="favicon.ico">
