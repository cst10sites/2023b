HTML Guide

Common HTML Tags outside <body>

Tag Definition Example
<!DOCTYPE html> Defines the document type. <!DOCTYPE html>
<html lang="en-CA"> Defines the language of the document. <html lang="en-CA">
<meta charset="UTF-8"> Defines which unicode is used to encode characters. Can encode 1,112,064 characters (Most languages and symbols).
<title></title> Creates a title for the document The tab of this website contains the title.
<style></style> The style tag creates internal CSS. <style>
body {
color: grey;
}</style>
This makes text in the body grey.
<link> Links the External CSS to this file <link rel="stylesheet" href="style.css">
<link> Links a favicon to this file <link rel="icon" type="image/x-icon" href="favicon.ico">
