Tag Definition Examples
<!DOCTYPE> Defines a language is HTML <!DOCTYPE>
<html lang = "en"> Defines a language is english <html lang = "en">
<meta charset = "UTF-8"> UTF-8 contained almost every character in the world <meta charset = "UTF-8">
<title> Defines a title of the website <title>This is the title</title>
<style> Allow to used the internal CSS <style> h1{ color: white; } </style>
<link> Called the href link <link rel = "example" href = "example.html">
<link> Can show the favicon <link rel = "icon" type = "image/x-icon" href = "favicon.ico">
