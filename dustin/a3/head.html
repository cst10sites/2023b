HTML Guide

Common HTML tags outside <body>

Tags Definitions Examples
<!DOCTYPE html> Defines document type <!DOCTYPE html>
<html lang="en"> Defines the language of the document <html lang="en">
<meta charset="UTF-8"> Defines which Unicode is used to encode characters All of the characters I use in this sentence and anywhere else on this site are part of UTF-8.
<title> </title> Defines the title of the document
<link> Used to define external CSS <link rel="stylesheet" href="styles.css">
<link> Defines a favicon <link rel="icon" type="image/x-icon" href="dizzy.ico">
<style> </style> Used to define internal CSS
