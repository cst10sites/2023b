Common html tags inside body

Head
Tags Definitions Examples
<p> Creates a paragraph. Useful for creating text inside a website. This is a paragraph
<h#> Replace the # with a number from 1-6 and it will create a heading in descending size. Useful for grabbing attention and insinuating a new section.

This is heading 4
<hr> Creates a horizontal line across a page. Useful for sectioning a website
&nbsp; Meaning Null Blank SPace (null and blank are the same word, why?), it creates a space equivalent to a space from the space bar. Like              this
<table>/<tr>/<th>/<td> All of these tags work together in order to make a table. <table> defines a table; <tr> makes a new row; <th> makes a header for a section of the table; <td> contains the data for a normal table row. This table that you're looking at. (duh)
<img src="" alt=""> Allows an image to be shown. Needs to have href and alt. The src being the source of the image and alt being a message that shows in case the image fails to load.
<a href="" target=""> Creates a hyperlink to another site. Two other things inside of the element: href and target. href is the site that it will go to and target depicts whether the link creates a new tab. Hyperlink to head
<ul> Creates an unordered list, also known as a bullet list
  • item 1
  • item 2
  • etc.
<ol> Creates an ordered list, also known as a numbered list
  1. item 1
  2. item 2
  3. etc.
<br> Meaning break, it creates a new line/space in a text.
