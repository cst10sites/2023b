HTML Guide

Common HTML Tags outside <body>

tag definition example(s)
<!DOCTYPE html> the tag to start an html file this file uses this tag!
<html lang="en"> set the html file language to english! this file uses this tag!
<meta charset="UTF-8"> tells the browser to use the utf-8 character encoding when translating machine code into human-readable text this file uses this tag!
<title></title> gives a title to the html file this file has a title!
<style></style> internal css is used to define styles for a specific html file ...
<link rel="stylesheet" type="text/css" href="styles.css"> external css is used to maintain constant styles throughout multiple html files this file uses external css!
<link rel="icon" type="image/png" href="favicon.png"> this tag is used to set a favicon for the webpage! this file uses a favicon!
