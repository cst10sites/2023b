HTML Tags

Common html Tags Inside <head>

Tags Definition Examples
<!DOCTYPE html> Tells the browser what type of file the website is.
<html lang="en"> Tells the browser that the original website is english.
<meta charset="UTF-8"> Allows the website to be translated into almost all languages.
<title></title> Title of website.
<style></style> Internal CSS. Designing the look of the website within the same code file.
<link rel="stylesheet" href="file"> <style></style> External CSS. Designing the look of the website on a seperate file. Connects to the code of the website through <style></style>
<link rel="icon" type="image/x-icon" href="file"> Favicon is the image next to the title.
