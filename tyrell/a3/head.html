Common HTML Tags outside <body>

Tags Definition Example
<!DOCTYPE html> This shows how the computer should display the information on the website. This website is in <!DOCTYPE html>
<html lang="en"> This changes what language the website in displayed in. This is english.
<meta charset="UTF-8"> This lets you use all sorts a different characters. •)!(@#&☻♀, and many more.
<title>...</title> This creates the title of the page located at the top of your screen. The title for this website is "Assignment 3".
body {} This is internal css, and it sets up how the website looks like the colour, background, fonts, etc. Body {color; white;} this will change the font colour too white.
<link rel="stylesheet" href="style.css"> This makes the website use the css in the "style.css" folder. Its useful for a website with more than one page. Body {color; white;} but this would be located in style.css.
<link rel="icon" type="image/x-icon" href="..."> This is what controls the fav icon, or in other words, the logo beside the title. The little guide books beside my title at the top of the screen.
