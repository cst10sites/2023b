Common HTML Tags Outside the <body>

Tag Definition Example
<!DOCTYPE html> Document type declaration.
<html lang="en"> Opens the HTML code and selects the language.
<meta charset="UTF-8"> Tells the browser to use the utf-8 character encoding.
<title></title> Title.
<style></style> Describes internal CSS. <style>
body {
background-color: black;
color: white;
} </style>
<link> Refers to external CSS. <link rel="stylesheet" href="style.css">
<link> Include favicon. <link rel="icon" type="image/x-icon" href="dizzy.ico">
