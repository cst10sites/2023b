Tag/Term Definition Examples
<!DOCTYPE html> This tag informs the browser that this document is an HTML All websites use this tag so that they function as websites
<html lang="en"> html lang sets the primary language for a HTML document. In this case, the "en" refers to "english" I used this tag to set this document's language to english
<meta charset="UTF-8"> This tag sets the UTF-8 character set, which covers almost all letters, numbers, and other characters All of our letters and numbers are part of the UTF-8 character set
<title> This tag names sites For example, this site's title is Assignment 3
Internal CSS Internal CSS styles are within the head of an HTML document and affect the entire document Internal CSS that are kept within <style> in the head section include backround-color
External CSS External syle sheets use one file to change the entire website's design External CSS files are defined by the <link> element within the head tag
Favicon Favicon is a file containing small icon(s) that are associated with a website An example is the small "a" next to the search bar on Amazon's website, as it is their logo
