Inside <body> guide

Common HTML Tags Outside <body>
Tag Description Example
<!DOCTYPE html> Declares the document type and version of HTML being used. <!DOCTYPE html>
<html> Represents the root element of an HTML document. <html></html>
<head> Contains metadata and other non-visible elements of the document. <head></head>
<title> Specifies the title of the HTML document. <title>My Page</title>
<meta> Provides meta-information about the HTML document. <meta charset="utf-8">
<link> Specifies external resources to be linked with the HTML document. <link rel="stylesheet" href="styles.css">
<style> Defines inline CSS styles for the HTML document. <style> body { background-color: #f5f5f5; } </style>
<script> Defines embedded or external JavaScript code for the HTML document. <script src="script.js"></script>
<noscript> Provides alternate content if JavaScript is disabled. <noscript>JavaScript is disabled.</noscript>
<base> Specifies the base URL for all relative URLs within the HTML document. <base href="https://www.example.com/">
<header> Represents the introductory or top section of a page or section. <header><h1>Welcome</h1></header>
